# React + Vite


In the project directory, you can install dependency by following command:

# `npm install`

In the project directory, you can run app by following command:

# `npm run dev`

Project will accessible on http://localhost:5174 with buttons to Add Company and Edit Compnany.

Kidnly first create company and than click on edit company.

Please create .env file with following contain:
VITE_APP_API_URL=http://localhost:3000/api
